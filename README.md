# Emotion_Awareness_using_Machine_Learning
This project mainly focuses on Emotiones of a person that is angry, sad, neutral, fear, surprise, disgust

download the project and ensure all files are in same folder and download the dataset from below link

Link to download dataset : https://www.kaggle.com/datasets/msambare/fer2013?resource=download

After downloading the dataset place it in the same folder as the project files and open your command prompt or terminal and enter the following commands

cd “Emotion_Awareness_using_Machine_Learning”
pip install streamlit streamlit-webrtc mediapipe av
streamlit run streamlit_app.py
